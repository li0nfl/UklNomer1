<img width="1280" height="797" alt="image" src="https://github.com/user-attachments/assets/555ecc7c-df0d-408c-a99b-e3095f253628" />Program yang saya buat digunakan untuk menghitung total biaya pengiriman paket berdasarkan:
	•	Berat paket (Kg)
	•	Jarak pengiriman (Km)
	•	Ukuran paket (panjang × lebar × tinggi dalam cm)
Program yang saya buat juga menambahkan biaya tambahan Rp 50.000 jika volume paket lebih dari 100 cm³. 

	1.	Pengguna memasukkan data paket → berat, jarak, panjang, lebar, dan tinggi.
	2.	Program menghitung volume
    3.	Menentukan biaya per kilogram:
	•	Jika jarak ≤ 10 km → Rp 4.250 per kg
	•	Jika jarak > 10 km → Rp 6.000 per kg
    4.  Menghitung biaya dasar
    5.	Memeriksa apakah ada biaya tambahan volume
    6.	Menjumlahkan semua biaya
    7.	Menampilkan hasil akhir berupa berat, jarak, volume, dan total biaya pengiriman
